Detailed Description

Outputs values to std::ostream conditionally.

Examples:

slim::element::conditional_ostream debug_log(std::cout)
debug_log << "it is printed." << std::endl;
slim::element::conditional_ostream debug_log(std::cout, false)
debug_log << "it is not printed." << std::endl;
Note
When the validity of a conditional ostream can be determined at compile time, the printing codes are usually omitted at runtime by compiler's optimization.
